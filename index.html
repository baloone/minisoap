Install dependecies

First you need to have python 3 and pip installed on your computer. You also will have to install ffmpeg on your computer. This can be done for instance on ubuntu just by running sudo apt install ffmpeg.

Then you will have to install the requirements by typing pip install -r requirements.tst

Launching the interpreter

To launch the interpreter you need to run python minisoap.py
